# impressive.js# * *impressive.js* is based on impress.js for impressive presentation * It uses and extends *builder4impress* for editing the slides * You can find a nice navigation menu at the top right * Here you can jump to any slide * You can use the *markdown* syntax
# Editing the slide positions # * Select the edit icon in the menu * Stop your pointer over a slide * Adjust the positioning to your liking * Chose *Get positions* to download a new position files * Overwrite the existing positions file with the new one
